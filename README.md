# ExploraMundo - Projeto de Certificação SENAI

Este site foi desenvolvido como projeto para o curso Jornada Dev no SENAI. A proposta era criar uma landing page funcional para uma agência de viagens, aplicando conceitos de semântica e estilização avançada.

## Sobre o desenvolvimento

A ideia do ExploraMundo foi criar uma interface que não apenas mostrasse destinos, mas que passasse uma sensação de sofisticação. Escolhi uma paleta de cores baseada em verde floresta e dourado e dar uma identidade visual mais robusta ao projeto.

Trabalhei bastante na parte de CSS para garantir que a navegação fosse fluida, utilizando `scroll-behavior: smooth` e filtros de imagem com `background-blend-mode`, o que permitiu manter os textos legíveis sobre as fotos sem precisar editar as imagens externamente.

## O que foi implementado

* **Estrutura Semântica:** O HTML foi montado pensando em acessibilidade e SEO, usando as tags corretas para cada parte do conteúdo.
* **Navegação Inteligente:** Menu fixo que acompanha o usuário e botões de retorno ao topo para facilitar a experiência.
* **Estilização Curada:** Foquei em detalhes como o sublinhado customizado nos títulos e efeitos de transição nos cards de destino.
* **Código Limpo:** As URLs das fotos foram simplificadas e o CSS foi organizado por seções para facilitar futuras manutenções.

## Tecnologias

Para este projeto, utilizei apenas HTML5 e CSS3 puro. A tipografia é a Segoe UI, que traz uma leitura limpa e moderna.
